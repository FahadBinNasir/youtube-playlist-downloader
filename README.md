# 🎥 YouTube Playlist Downloader (C# WinForms)

A simple Windows desktop application built with C# WinForms using yt-dlp.

Supports downloading:
- YouTube videos
- Playlists
- Audio extraction (MP3)
- Subtitle download

# YTDownloader
A YT Full Playlist or a Single Video with Subtitles Downloader 


## 📦 Requirements

This app requires FFmpeg.

Download FFmpeg here:
https://www.gyan.dev/ffmpeg/builds/

OR 
Where I Downloaded it From :
https://ffmpeg.en.uptodown.com/windows/download

After downloading:
- Extract zip
- Place `ffmpeg.exe` in the same folder as YTDownloader.exe
